# Contributing to Barium

Thank you for your interest in contributing to Barium!

## Forking the Repository
1. Click the "Fork" button at the top right of the repository page.
2. This will create a copy of the repository in your GitHub account.

## Cloning the Repository
1. Open your terminal or command prompt.
2. Run the following command to clone the repository:
   ```bash
   git clone https://github.com/YourUsername/Barium.git
   ```
3. Replace `YourUsername` with your GitHub username.

## Creating Feature Branches
1. Navigate into your cloned repository:
   ```bash
   cd Barium
   ```
2. Create a new branch for your feature:
   ```bash
   git checkout -b feature/your-feature-name
   ```

## Code Guidelines for Mixins
- Follow the existing coding standards and indentation in the repository.
- Write clear and concise mixin names that describe their purpose.
- Document each mixin's functionality and parameters in the comments.

## Testing Procedures
- Ensure to write tests for any new feature.
- Run tests using the following command:
  ```bash
  npm test
  ```
- Confirm all tests pass before submitting your Pull Request (PR).

## Reporting Issues
- If you encounter any issues, please report them by opening an issue in the repository with a clear description of the problem and steps to reproduce it.

## Code of Conduct
- Please be respectful and inclusive to all contributors.
- Harassment of any kind will not be tolerated.
- For more details, please refer to our Code of Conduct document in the repository.

## Need Help?
- If you're stuck or have questions, feel free to ask for help by opening an issue or by reaching out to the contributors directly.
  
---

Thank you for contributing! Your efforts help make Barium a better project for everyone!
