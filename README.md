# Barium Mod Documentation

## What is Barium?
Barium is a powerful mod designed to enhance player experience and optimize game performance in Minecraft. It integrates various features that target specific areas of gameplay to provide a smoother and more efficient gaming environment.

## What Does Barium Do?
Barium works to improve the overall performance of Minecraft by implementing four main optimization modules, each aimed at reducing lag, enhancing gameplay speed, and optimizing resource usage.

## Optimization Modules
1. **AI Brain Throttling**  
   This module smartly manages the AI processing, ensuring that mobs and NPCs operate efficiently without consuming excessive resources.

2. **Fast Leaf Decay**  
   This feature accelerates the decay of leaves when trees are chopped down, reducing server lag caused by lingering leaves.

3. **RAM Optimization**  
   Reduces memory usage while playing, allowing the game to run smoother, especially on lower-end machines.

4. **Math Optimization**  
   Streamlines mathematical calculations performed by the game, thereby improving performance during complex computations.

## Technical Specifications
- **Game Version Compatibility:** 1.21 and above  
- **Operating System Compatibility:** Windows, MacOS, Linux, Android
- **Required RAM:** Minimum 1GB (4GB recommended)
- **Processor:** Dual-core CPU or better

## Installation Instructions
1. Download the latest version of the Barium mod from the official repository.  
2. Navigate to your Minecraft `mods` directory.  
   - On Windows: `C:\Users\YourUsername\AppData\Roaming\.minecraft\mods`  
   - On MacOS: `~/Library/Application Support/minecraft/mods`  
   - On Linux: `~/.minecraft/mods`
   - On Android click `import mod`
3. Place the downloaded Barium .jar file into the `mods` directory.
4. Launch Minecraft with the Fabric instance to enjoy the enhanced experience!
